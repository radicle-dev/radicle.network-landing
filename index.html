Getting started with the CLI

  1. Download the Radicle CLI here.
  2. Run rad auth to create your Radicle identity.

    This authenticates you and gives you a device key pair.

  3. Navigate to an existing git repo on your machine.
  4. Run rad init to create a Radicle project from the repo.

    Ideally no name is specified here or your password is used (WHAT DOES THIS MEAN?????)

  5. Run rad push to publish to the network
  6. View and share your Radicle project from a URL on app.radicle.network.

    For example: https://app.radicle.network/seeds/seed.cloudhead.io/projects/rad:git:hnrkqi6ohci9m59i54ppiy3fqkedkjt99ymdo
